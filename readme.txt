Exercise 1 (To Do List)

Objective: Build a basic to-do list where users can add and remove tasks.

Step 1: Set Up the HTML Structure
Create a new HTML file (e.g., index.html) in your text editor.
Define the basic structure of your HTML document, including a title, a container for the to-do list, an input field, a button to add tasks, and an empty list to display tasks.

Step 2: Create CSS Styles
Create a new file called styles.css in the same directory as your HTML file.
Style your to-do list elements using CSS to make it visually appealing. 

Step 3: Write JavaScript Logic
Create a new file called script.js in the same directory as your HTML file.
Implement JavaScript code to handle adding and removing tasks in your to-do list.
 
Step 4: Test Your To-Do List
Interact with your to-do list by adding tasks through the input field and clicking the "Add" button. You should also be able to add tasks by pressing the Enter key.
To remove a task, click the "Delete" button next to it.

Excercise 2 (Add to Cart)

Objective: Build a basic "Add to Cart" functionality for an online store.

Step 1: Set Up the HTML Structure
Create an HTML file (e.g., index.html) in your text editor.
Design the basic structure of your online store, including product listings, a cart section, and a way to display the items added to the cart.
Each product listing should have a name, price, and an "Add to Cart" button.

Step 2: Create CSS Styles
Create a new file called styles.css in the same directory as your HTML file.
Style your online store elements using CSS to make it visually appealing. Define styles for product listings, the cart section, and any other relevant components.

Step 3: Write JavaScript Logic
Create a new file called script.js in the same directory as your HTML file.
Implement JavaScript code to handle the "Add to Cart" functionality.
When a user clicks the "Add to Cart" button for a product, the selected item should be added to a cart object.
Maintain a cart object that stores items with details such as name, price, and quantity.
Update the cart's content dynamically when items are added.
Allow users to remove items from the cart.

Step 4: Display Cart Information
Create a section in your HTML to display the cart contents, including the names, quantities, and total price of items in the cart.
Update this cart display whenever items are added or removed.

Step 5: Test Your "Add to Cart" Feature
Add items to the cart by clicking the "Add to Cart" buttons.
Verify that the cart content is updated correctly.
Implement the ability to remove items from the cart and ensure the cart total updates accordingly.

Step 6: Additional Enhancements (Optional)
Consider adding features such as updating the cart quantity, calculating the total price, or providing a checkout process.
You can also style the cart display to make it more user-friendly.
